# video2bas

#### 介绍
通过ffmpeg,将视频转换成为BAS弹幕代码

#### 软件架构

php + ffmpeg

#### 使用说明

1.  把待转换的视频Copy到genertor.php同目录下
2.  确保在genertor.php同目录下,存在bas、jpg、txt三个文件夹
3.  修改genertor.php中的$videoName为待转换视频的名字
4.  运行generator.php即可,BAS弹幕代码将会保存在bas文件夹中
